## Run database:
  - copy `.env.example` into `.env`
  - `docker compose up -d`

## Run frontend:
  - copy `.env.example` into `.env`
  - `npm i`
  - `npm run dev`

## Run backend:
  - copy `.env.example` into `.env`
  - `npm i`
  - `npm run dev`

## Domain example:
  - `http://localhost:3000/quizzes`
